Automated build tools like NAnt can streamline your build process and alleviate some of the headaches of team integration. But using these tools to build a Setup and Deployment project is another story its impossible to build a setup project using csc, and the MSI task in NAnt leaves much to be desired.

Enter WiX, an XML-based, scriptable, toolset originally developed by Microsoft that can be used to automate the build of an MSI. Using WiX, part of the output from your build can be an up-to-date installer for your product.

While extremely powerful and flexible, WiX is also somewhat of an enigma. Its fairly complex and not very well documented. But not to worry Microsoft installer guru Rob Mensching has published an excellent article on MSDN explaining how WiX works and how you can use it in your build process.
